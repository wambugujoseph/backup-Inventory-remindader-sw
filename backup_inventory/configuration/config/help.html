2. Fill in all the details.

4. Resart the application

Inventory generation

Daily Inventory
 1. On the right hand side QUERY OPTIONS section, select the date which you want to generate inventory on
2. Click View button to display the result
3. To print the result. On the display area top left click "print preview" button
Departmental Inventory
 Creates an inventory for all the backups that the dapertment has ever made.
1. On the right hand side QUERY OPTIONS section, select the department in the Combo Box.
Note(ensure you do not select the from and to date, leave them empty)
2. Click the "View" button in that section to display the results
3. To print the result. On the display area top left click "print preview" button
Departmental Inventory From,to a specified date
1. On the right hand side QUERY OPTIONS section, select the department in the Combo Box.
2. Select the date From(starting date) To(end date)
2. Click the "View" button in that section to display the results
3. To print the result. On the display area top left click "print preview" button
Daily Inventories For specified period of time
 1. Click the "print backup inventory" button on the top right hand
2. Select the form of grouping you want (date or department)
(a). Group By date: groups backup detail made on each date.
(b). Group By Department: groups backup details for each department based on the stated time.
3. Click the Print previw to see the results
